**Weapon Arts** are special techniques that the Player can use with a variety of effects. While most deal damage, some allow the user to :FighterArts_defense: shield themselves or :FighterArts_heal: heal the party. Some are stronger against a certain type of enemy, and some become more effective if an enemy is hit from a certain direction.

Arts with a **special effect** are indicated by a smaller symbol in the lower half of the icon:

|           Icon            | Description                                                            |
| :-----------------------: | ---------------------------------------------------------------------- |
|  :WeaponArts_art_front:   | Extra Damage if attacking from the enemy's front.                      |
|   :WeaponArts_art_rear:   | Extra Damage if attacking from the enemy's back.                       |
|   :WeaponArts_art_side:   | Extra Damage if attacking from the enemy's side.                       |
|  :WeaponArts_art_cancel:  | Increases Damage ratio after canceling an Weapon-Attack.               |
| :WeaponArts_art_species:  | Does extra damage to enemies of a certain type (e.g. aquatic enemies). |
| :WeaponArts_art_recharge: | Critical hits will recharge the art used.                              |
There are 3 Weapon Art Slots Given on a Signature Blade, Each can be unlocked as the Campaign Progresses.

| Level | Description           |
| :---: | --------------------- |
| Lv. 1 | Access to one Art.    |
| Lv. 6 | Access to two Arts.   |
| Lv. 9 | Access to three Arts. |
